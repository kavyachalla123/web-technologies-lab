# React Application for Displaying Multiple Student Cards

## 1. Introduction
This project is a React-based web application that displays multiple student records using reusable components. Each student’s details such as name, department, and marks are shown using a card layout. The application demonstrates component reusability and props in React.

---

## 2. Objective
The main objectives of this application are:
- To understand component reusability in React
- To pass data between components using props
- To display dynamic data in multiple components
- To structure applications using parent-child components

---

## 3. Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

---

## 4. Functional Requirements Implemented
- Creation of a reusable student card component
- Passing data from parent component using props
- Displaying student details dynamically using props
- Rendering multiple components with different data
- Maintaining component hierarchy using parent and child components
- Structuring UI using JSX elements

---

## 5. Application Flow
1. Parent component stores student data
2. Data is passed to child component using props
3. Child component displays student information
4. Multiple instances of component are rendered
5. UI updates based on props provided

---

## 6. How to Run the Application

1. Navigate to project directory:
   ```bash
   cd exercise-2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

4. Open browser:
   ```
   http://localhost:3000
   ```

---

## 7. Conclusion
This project demonstrates component reusability and data passing using props in React. It helps in understanding how multiple UI elements can be generated efficiently using reusable components.
