# React Application for Displaying Student Profile

## 1. Introduction
This project is a React-based web application developed to display the profile details of a student. The application presents information such as name, department, year, and section in a structured format using React components and JSX.

---

## 2. Objective
The main objectives of this application are:
- To understand the structure of a React application
- To implement functional components in React
- To use JSX for building user interfaces
- To display dynamic data using JavaScript variables
- To understand component rendering in React

---

## 3. Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

---

## 4. Functional Requirements Implemented
- Creation of a functional component to display student details
- Use of JSX for combining HTML structure with JavaScript
- Storage of student data using variables inside the component
- Dynamic rendering of values using JSX expressions
- Structuring content using headings and paragraph elements
- Exporting and importing components
- Rendering component within the main application

---

## 5. Application Flow
1. Student data is defined using JavaScript variables
2. JSX is used to structure the UI
3. Data is dynamically displayed using expression binding
4. Component is rendered inside the App component
5. UI is displayed through the root component

---

## 6. How to Run the Application

1. Navigate to project directory:
   ```bash
   cd exercise-1
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

4. Open browser:
   ```
   http://localhost:3000
   ```

---

## 7. Conclusion
This project demonstrates the basics of React including component creation, JSX usage, and dynamic data rendering. It provides a clear understanding of how user interfaces are built using React.
