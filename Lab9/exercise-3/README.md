# React Application for Counter Using useState Hook

## 1. Introduction
This project is a React-based web application that implements a simple counter system. The application allows users to increase or decrease a numeric value using buttons. It demonstrates state management and user interaction in React.

---

## 2. Objective
The main objectives of this application are:
- To understand state management using useState Hook
- To implement dynamic UI updates in React
- To handle user interaction through event handling
- To build interactive components using React

---

## 3. Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

---

## 4. Functional Requirements Implemented
- Creation of a functional component for counter
- Use of useState Hook to store counter value
- Initialization of state with default value
- Display of dynamic counter value
- Implementation of increment and decrement functionality
- Handling button click events using onClick
- Updating state using state updater function
- Automatic UI re-rendering after state change

---

## 5. Application Flow
1. Counter value is initialized using useState
2. User clicks increment or decrement button
3. Event handler updates state value
4. Component re-renders automatically
5. Updated value is displayed in UI

---

## 6. How to Run the Application

1. Navigate to project directory:
   ```bash
   cd exercise-3
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

4. Open browser:
   ```
   http://localhost:3000
   ```

---

## 7. Conclusion
This project demonstrates how to manage state and handle user interaction in React. It highlights the importance of useState Hook in building dynamic and interactive applications.
