# React Application for Form Validation

## 1. Introduction
This project is a React-based web application developed to collect user information through a form. The application validates user inputs such as name, email, and password before submission. It demonstrates the concept of controlled components and form validation in React.

---

## 2. Objective
The main objectives of this application are:
- To understand controlled components in React
- To manage form data using useState Hook
- To implement input validation using conditional logic
- To display dynamic error messages in the user interface
- To handle form submission and reset functionality

---

## 3. Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

---

## 4. Functional Requirements Implemented
- Creation of a functional component for form handling
- Use of useState Hook to manage form input fields
- Binding of input values using controlled components
- Handling user input through onChange event
- Validation of input fields (name, email, password)
- Displaying validation error messages dynamically
- Preventing form submission when validation fails
- Handling form submission using onSubmit event
- Resetting form fields after successful submission

---

## 5. Application Flow
1. User enters input values in the form fields
2. onChange event updates state variables
3. On form submission, validation logic is executed
4. Errors are displayed if validation fails
5. If validation is successful, form is submitted
6. Form fields are reset after submission

---

## 6. How to Run the Application

1. Navigate to project directory:
   ```bash
   cd exercise-1
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

4. Open browser:
   ```
   http://localhost:3000
   ```

---

## 7. Conclusion
This project demonstrates how to implement form handling and validation in React using controlled components and hooks. It ensures proper user input validation and enhances user interaction with dynamic feedback.
