# React Application for Dynamic List Management

## 1. Introduction
This project is a React-based web application developed to display and manage a dynamic list of items. The application allows users to add and remove items from the list. It demonstrates list rendering and key management in React.

---

## 2. Objective
The main objectives of this application are:
- To understand list rendering in React
- To manage multiple items using array state with useState
- To implement dynamic addition and removal of list items
- To use unique keys for efficient rendering
- To handle user interaction through event handling

---

## 3. Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

---

## 4. Functional Requirements Implemented
- Creation of functional components for input and list display
- Use of useState Hook to manage list of items
- Rendering list dynamically using map() function
- Assigning unique keys to each item
- Adding new items through user input
- Removing items using button click events
- Updating UI dynamically after state changes
- Handling empty list scenarios using conditional rendering
- Maintaining separation between input and display components

---

## 5. Application Flow
1. User enters a new item in the input field
2. Item is added to the list using state update
3. List is rendered dynamically using map()
4. Each item is assigned a unique key
5. User can remove items using the remove button
6. UI updates automatically after each operation

---

## 6. How to Run the Application

1. Navigate to project directory:
   ```bash
   cd exercise-2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```

4. Open browser:
   ```
   http://localhost:3000
   ```

---

## 7. Conclusion
This project demonstrates dynamic list rendering and efficient state management in React. It highlights the use of keys for optimized rendering and showcases user interaction through add and remove operations.
